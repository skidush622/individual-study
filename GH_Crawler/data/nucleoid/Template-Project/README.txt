-Razor View Engine
-Sharp Architecture
-NHibernate 3
-FluentNHibernate
-FluentMigrator
-Autofac (instead of Windsor)
-RhinoMocks
-MBUnit (instead of NUnit)
-NLog (instead of log4net)
-Quartz.Net
